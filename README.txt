PORTFÓLIO — INSTRUÇÕES DE PUBLICAÇÃO

1. Abra o Gerenciador de Arquivos do cPanel.
2. Entre na pasta public_html do domínio.
3. Envie todo o conteúdo desta pasta (index.html e a pasta assets).
4. Confirme que index.html está diretamente dentro de public_html.
5. Acesse o domínio pelo navegador. Nenhuma instalação ou configuração é necessária.

O site usa somente HTML5 e CSS3. Não depende de JavaScript, PHP, banco de dados,
fontes externas, instalação ou compilação.

Para alterar textos, edite index.html.
Para alterar cores ou layout, edite assets/style.css.
Para trocar a foto, substitua assets/profile-photo.png e ajuste somente
"object-position" no bloco ".avatar img" do CSS, se necessário.
